hexo常用指令

发表于 2019-02-16
本文字数： 212 阅读时长 ≈ 1 分钟

1.初始化

$ hexo init [folder]

2.新建一篇文章。

$ hexo new [layout] <title>

3.生成静态文件。

$ hexo generate该命令可以简写为$ hexo g

4.部署网站

$ hexo deploy该命令可以简写为$ hexo d

5.监视文件变动

$ hexo -w, --watch

6.server

$ hexo server

启动本地服务器。默认情况下，访问网址为： http://localhost:4000/。
